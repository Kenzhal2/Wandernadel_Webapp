Map View - Responsive Test

Test Requirements

Verify the map mockup is responsive and location card displays correctly at:

  • 320px (Mobile) - Location card at bottom, shows top 120px, expandable
  • 768px (Tablet) - Location card as side panel (400px width) on right
  • 1024px (Desktop) - Location card side panel (450px width)
  • 1920px (Large Desktop) - Location card side panel (500px width)

How to Test:

  1. Open browser DevTools (F12)
  2. Enable responsive design mode (Ctrl+Shift+M / Cmd+Shift+M)
  3. Test each breakpoint: 320px, 768px, 1024px, 1920px
  4. Verify the following:
    • Map fills available viewport height
    • Location card displays correctly at each breakpoint
    • Card is expandable/collapsible on mobile
    • Card appears as side panel on tablet+
    • Touch targets are ≥44px
    • No horizontal scrolling
    • Markers are clickable and visible
    • User location indicator is visible
  5. Test on actual mobile device if possible

Mobile (320px) Specific Tests:

  • ✅ Location card appears at bottom, showing top 120px
  • ✅ Card can be expanded by clicking toggle button
  • ✅ Card scrolls smoothly when expanded
  • ✅ Card doesn't overlap bottom navigation
  • ✅ Map markers are large enough to tap (≥44px)
  • ✅ Header buttons are touch-friendly
  • ✅ Collect stamp button is full-width and accessible
