Passport View - Responsive Test

Test Requirements

Verify the passport mockup is responsive at the following breakpoints:

  • 320px - Mobile (small phones) - 2 columns, 100px stamps
  • 768px - Tablet - 3 columns, 140px stamps
  • 1024px - Desktop - 4 columns, 160px stamps
  • 1920px - Large Desktop - 5 columns, 180px stamps

How to Test:

  1. Open browser DevTools (F12)
  2. Enable responsive design mode (Ctrl+Shift+M / Cmd+Shift+M)
  3. Test each breakpoint by setting the viewport width to: 320px, 768px, 1024px, and 1920px
  4. Verify the following:
    • Stamp grid adjusts column count correctly
    • Stamp sizes scale appropriately
    • Progress bar remains readable
    • Bottom navigation is accessible
    • Text remains readable at all sizes
    • No horizontal scrolling occurs
  5. Alternatively, resize your browser window to test fluid responsiveness
